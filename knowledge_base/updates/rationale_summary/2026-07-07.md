# Rationale Summary: 2026-07-07

## Goal

尽量将公开电网数据集缓存到本地，并把不能无凭证全量下载的数据源转成可追踪、可继续执行的本地入口。

## Result

- 直接下载或 Git 缓存的数据源增加到 12 个。
- 所有登记数据源都已有本地路径；manifest 中不再保留空路径。
- ENTSO-E、PJM、EIA、NREL NSRDB、ACN-Data、Zenodo TB 级数据保留为 metadata-only/API-ready，因为它们需要 token/API key、账号、endpoint 选择或 TB 级子集策略。

## Key Constraints

- ENTSO-E Transparency API 需要 security token。
- ACN-Data API 需要注册 token。
- NREL NSRDB 和 EIA Open Data 正式批量拉取需要 API key。
- Zenodo large synthetic power-grid ML dataset 总量 TB 级，不适合初始化时全量下载。
- PJM Data Miner 2 可公开访问说明页，但数据批量获取需要按 endpoint/API 流程进一步配置。
