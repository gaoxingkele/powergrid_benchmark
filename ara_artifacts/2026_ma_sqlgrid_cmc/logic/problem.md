# Problem

## Research Question

Can compact domain-grounded context construction and reference-free validation improve Text-to-SQL execution accuracy for a fixed power-grid maintenance database?

## Benchmark Task

Natural-language power-grid maintenance questions are mapped to executable SQLite SQL queries.

## Inputs

- Natural-language question.
- Fixed database schema and local value inventory.
- Optional compact domain context and validation signals depending on condition.

## Outputs

Read-only SQL query and execution result scored against benchmark annotations.
